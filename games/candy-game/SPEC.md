# 三消游戏规范文档

## 1. 项目概述
- **项目名称**: 糖果消消乐 (Candy Crush Clone)
- **类型**: HTML5 三消类休闲游戏
- **核心功能**: 经典三消玩法，每关包含3个阶段，难度递增
- **目标用户**: 休闲游戏玩家

## 2. UI/UX 规范

### 布局结构
- **画布尺寸**: 480px × 720px (手机竖屏比例)
- **游戏区域**: 8×8 网格，每个格子 50px × 50px
- **顶部信息栏**: 显示关卡、分数、目标、阶段
- **底部控制**: 步数限制提示

### 视觉设计
- **主色调**:
  - 背景: 渐变紫 (#1a1a2e → #16213e → #0f3460)
  - 强调色: 霓虹粉 (#ff2e63), 霓虹蓝 (#08d9d6), 霓虹黄 (#f9ed69)
- **糖果颜色** (6种):
  - 红色: #ff6b6b (红苹果)
  - 橙色: #ffa502 (橙子)
  - 黄色: #ffeaa7 (香蕉)
  - 绿色: #55efc4 (青柠)
  - 蓝色: #74b9ff (蓝莓)
  - 紫色: #a29bfe (葡萄)
- **字体**: 'Fredoka One', cursive (圆润可爱风格)
- **动画效果**:
  - 消除: 缩放 + 淡出 (0.3s)
  - 下落: 弹性动画 (0.4s)
  - 选中: 弹跳效果
  - 阶段转换: 闪光动画

### 组件
- 糖果格子: 圆角矩形 + 糖果图标
- 选中状态: 发光边框 + 轻微放大
- 消除特效: 粒子爆炸效果
- 阶段进度条: 3个分段显示
- 按钮: 圆角 + 渐变 + 悬停动画

## 3. 功能规范

### 核心玩法
- 交换相邻两个糖果
- 3个或更多相同糖果连成一线消除
- 消除后上方糖果下落，新糖果从顶部生成

### 关卡阶段设计
每关包含3个阶段：

**第一阶段 - 入门** (简单)
- 目标分数: 500分
- 步数: 15步
- 特点: 大量简单匹配机会，5-6种糖果类型
- 完成率: ~95%

**第二阶段 - 进阶** (中等)
- 目标分数: 1500分
- 步数: 12步
- 特点: 需要策略思考，5种糖果类型，可能需要连锁消除
- 完成率: ~60%

**第三阶段 - 炼狱** (极难)
- 目标分数: 3000分
- 步数: 10步
- 特点: 只有4种糖果类型，需要精打细算，多重连锁
- 完成率: ~10% (90%的人完成不了)

### 游戏机制
- 步数用完且未达标 = 失败
- 阶段失败可重新挑战该阶段
- 三个阶段都通过 = 通关

### 视觉效果
- 霓虹发光效果
- 消除时屏幕轻微震动
- 连击时显示 Combo 文字
- 完成阶段时庆祝动画
